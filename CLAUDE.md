# Projeto: Sá Pires Warriors

Este projeto utiliza o Antigravity e Claude Code com skills customizadas para desenvolver um jogo RPG web completo. 
**Importante:** Nunca sobrescreva os arquivos originais em `_assets_original_backup`. Utilize sempre as cópias organizadas na pasta `public/`.
Todo o desenvolvimento deve ser guiado pelo documento `docs/PRD.md`.
Certifique-se de executar `npm run lint` e `npm run build` antes de declarar qualquer sprint concluída.
Nunca exponha as variáveis de ambiente sensíveis. Leia `docs/PRD.md` seção 20.
